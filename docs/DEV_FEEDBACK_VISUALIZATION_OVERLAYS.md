## 颜色约定

预测/人工 mask 对比图采用固定颜色语义：

| 区域 | 颜色 | 含义 |
| --- | --- | --- |
| 仅预测 mask | 绿色 | 算法认为是肺部，但人工标注没有覆盖 |
| 仅人工 mask | 蓝色 | 人工标注认为是肺部，但算法漏掉 |
| 二者重叠区域 | 黄色 | 预测与人工标注一致 |

连通域伪彩色图使用确定性调色规则，同一个输入每次生成的颜色一致，便于报告截图和复现实验。

## 使用方式

单张或单个 3D 切片：

1. 打开 CT 图像或 NIfTI 体数据。
2. 执行 `处理 -> 一键肺部分割`。
3. 用 `视图 -> 显示连通域伪彩色图` 查看阈值候选连通域分布。
4. 打开人工 mask 并执行 `分析 -> 计算 Dice / IoU`。
5. 用 `视图 -> 显示预测/人工对比图` 查看误差位置。

批处理：

- `批量处理当前数据...` 和 `一键处理配置数据集...` 均会在 `overlays/` 中额外输出可视化图。
- 这些图片可以直接用于报告和 PPT 的处理流程展示。

## 当前能力边界

- 该阶段只做静态叠加图，没有增加交互式透明度调节。
- 颜色语义基于 OpenCV BGR 输出，但保存为普通图片后在查看器中会显示为预期颜色。
- 医学左右仍沿用图像坐标左右；报告中需要继续注明这一点。
