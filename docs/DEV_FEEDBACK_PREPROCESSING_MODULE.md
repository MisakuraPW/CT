  - `处理 -> 灰度/归一化`

  - `处理 -> 中值滤波`

  - `视图 -> 显示高斯滤波结果`
  - `视图 -> 显示中值滤波结果`
  - `视图 -> 显示 CLAHE 增强结果`

## 设计说明

- NIfTI 读取模块保留原始切片的数据类型，因此肺窗可以对 16-bit/float CT 切片执行窗口映射。
- 对普通 8-bit PNG/TIFF/JPG 图像，灰度/归一化、滤波和 CLAHE 仍然可以正常运行。
- 预处理结果与分割结果一样可通过 `保存当前结果` 导出为图片，用于报告和 PPT。
- 切换 3D 体数据切片时，会清除当前切片的预处理派生结果，避免误看上一切片的缓存。

## 当前能力边界

- 预处理参数暂时使用固定默认值，没有做参数对话框。
- 肺窗默认按肺实质常用窗口 `WL=-600, WW=1500` 生成；其他窗宽窗位后续可加配置或输入框。
- 分割流程内部仍使用原有默认灰度化和高斯+Otsu 流程，预处理菜单主要用于独立展示和结果保存。

## 下一阶段建议

- 增加一个参数设置对话框，允许调整窗宽窗位、滤波核大小和 CLAHE 参数。
- 增加后台批处理进度 UI，避免处理完整数据集时主窗口长时间无响应。
